# Spelarbilder (för Bonus-vyn)

Lägg in en JPG/PNG per stjärnspelare. Filnamnet är slugifierat namn,
exempelvis:

- `kylian-mbappe.jpg`
- `lamine-yamal.jpg`
- `vinicius-junior.jpg`
- `jude-bellingham.jpg`
- `harry-kane.jpg`
- `erling-haaland.jpg`
- `gianluigi-donnarumma.jpg`
- `unai-simon.jpg`
- `mike-maignan.jpg`

Slug-regel: gemener, å/ä/ö -> a/a/o, accenter borttagna, mellanslag -> bindestreck,
inget annat än [a-z0-9-].

Saknas bilden visas initial-bricka automatiskt. Kvadratiska bilder ~400x400 px.
